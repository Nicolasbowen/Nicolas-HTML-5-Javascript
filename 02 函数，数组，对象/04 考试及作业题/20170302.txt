vux 这个组件库并不好用


 vux2必须配合vux-loader使用, 请在build/webpack.base.conf.js里参照如下代码进行配置：

const vuxLoader = require('vux-loader')
module.exports = vuxLoader.merge(webpackConfig, {
  options: {},
  plugins: [
    {
      name: 'vux-ui'
    }
    ]
})


这个配置时，用变量webpackConfig接受文件中 module.exports 右侧内容
然后加入以上代码
